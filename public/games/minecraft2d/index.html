After the game loads, you'll see menus:

  1. Click "Gotcha!" on the disclaimer screen
  2. Click "Play Game" on the main menu
  3. The game will start!

Controls:

  • Left Click: Place block (when holding a block)
  • Ctrl: Toggle between place/break mode
  • A/D: Move left/right
  • Space: Jump (disabled)
  • 1-2: Select block type (Grass, Dirt)
  • Shift: Crouch
