# LittleJS Quest - Platformer Game

A retro-style 2D platformer game powered by the LittleJS game engine.

## Features

- **Classic Platformer Gameplay**: Jump, run, and explore levels
- **Collectibles**: Gather coins and find secrets
- **Enemies**: Avoid or defeat enemies in your path
- **Destructible Terrain**: Break crates and interact with the environment
- **Smooth Physics**: Responsive controls and realistic platforming physics
- **Touch Support**: Play on mobile devices with touch controls

### Gamepad Support
- **Left Stick**: Move
- **A Button**: Jump
- **X Button**: Shoot

## Gameplay Tips

1. **Coins**: Collect yellow coins to increase your score
2. **Crates**: Break wooden crates by jumping on them
3. **Enemies**: Avoid red enemies or shoot them if you have a weapon
4. **Checkpoints**: Your progress is saved at certain points
5. **Deaths**: Deaths are tracked - try to complete levels without dying!

## Optimization

This game has been optimized for performance:
- Reduced particle effects for better mobile performance
- Auto-scaling canvas for responsive display
- Efficient rendering for low CPU usage
- Touch controls optimized for mobile devices

## Technical Details

- **Engine**: LittleJS v1.17.10
- **Platform**: Web (HTML5 Canvas)
- **Controls**: Keyboard, Mouse, Touch, Gamepad
- **Resolution**: Auto-scales to screen size
- **Performance**: Optimized for 512MB RAM / 0.15 CPU
